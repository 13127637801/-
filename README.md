#### 介绍

#### 软件
软件架构说明


#### 安装教程 可直接运行
1.  git clone https://gitee.com/geekzhang6/wechat-code-scanning-order.git
2.  cd ./wechat-code-scanning-order/wxserver 
3.  npm install
4.  npm start
